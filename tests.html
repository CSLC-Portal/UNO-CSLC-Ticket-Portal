Tests for Portal Functionality

The unit tests that we have in place for testing all of the CSLC portal functionality utilize the PyTest module. PyTest is a framework for writing varius tytpes of tests that we have in place such as unit tests, integration tests, and end-to-end tests that involve every aspect of the CSLC portal. Below we will outline the framework and what the indivudal tests do.
